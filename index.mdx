## What is Costume Retail System?

Costume Retail System is a full-stack SaaS platform designed to streamline costume rental operations. Built with NestJS and Astro, it provides a modern, efficient solution for managing costume inventory, tracking rentals, handling customer relationships, and automating laundry workflows.

## Technology Stack

Built with modern, production-ready technologies:

- **Backend**: NestJS with TypeScript
- **Frontend**: Astro
- **Database**: PostgreSQL with TypeORM
- **API Documentation**: Swagger/OpenAPI
- **Containerization**: Docker support
